# Moss & Associates Attorneys – Website Redesign

## Project Overview

This project is a complete redesign of the Moss & Associates Attorneys website, developed as part of the Melsoft Academy Company Project 1.

The goal of this project was to modernize the firm's online presence by improving the user experience, visual design, accessibility, and responsiveness while maintaining the company's professional branding and identity.

---

## Company

**Moss & Associates Attorneys**

**Industry:** Legal Services

**Original Website:** *https://www.mossinc.co.za/*

---

## Project Objectives

The redesign focuses on:

- Modern and professional user interface
- Improved user experience (UX)
- Fully responsive design
- Clear navigation
- Better visual hierarchy
- Stronger calls-to-action
- Improved accessibility
- Faster and cleaner user journey

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate into the project

```bash
cd moss-redesign
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```

---

## Production Build

Build the application

```bash
npm run build
```

Run the production server

```bash
npm start
```
